Test _ the value if the value of unity which comes to expression, then the associated value is returned. (SSN, GPA)

WHEN 'A' THEN 4

  1. User-Defined Functions

Oracle PL / SQL functions are available on the Oracle SQL statement. In Microsoft SQL Server, you can get this feature usually use a different method.

In the example below, using the Oracle user-defined function of GET_SUM_MAJOR indicates the sum of the fees that students pay. In SQL Server, you can do so by using a query as a table.

Oracle

Microsoft SQL Server

SELECT SSN, FNAME, LNAME, ) TUITION_PAID,
TUITION_PAID/GET_SUM_MAJOR(MAJOR)
AS PERCENT_MAJOR
FROM STUDENT_ADMIN.STUDENT

SELECT SSN, FNAME, LNAME, TUITION_PAID, TUITION_PAID/SUM_MAJOR AS PERCENT_MAJOR
FROM STUDENT_ADMIN.STUDENT,
(SELECT MAJOR, SUM(TUITION_PAID) SUM_MAJOR
FROM STUDENT_ADMIN.STUDENT
GROUP BY MAJOR) SUM_STUDENT
WHERE STUDENT.MAJOR = SUM_STUDENT.MAJOR

CREATE OR REPLACE FUNCTION GET_SUM_MAJOR
(INMAJOR VARCHAR2) RETURN NUMBER
AS SUM_PAID NUMBER;
BEGIN
SELECT SUM(TUITION_PAID) INTO SUM_PAID
FROM STUDENT_ADMIN.STUDENT
WHERE MAJOR = INMAJOR;
RETURN(SUM_PAID);
END GET_SUM_MAJOR;

You do not need the CREATE FUNCTION statement. Use the CREATE PROCEDURE statement.

  1. Pattern matching

The Oracle of SQL Server LIKE keyword offers useful wildcard search options that are not supported. SQL Server also supports a common support the% and _ wildcard characters in both the RDBMS as well as the [] and [^] characters.

[] Character set is used to search for a single character in the specified range. For example, you can specify as LIKE '[af]' or LIKE '[abcdef]' If you search for text in a single character position from up to f a. Below you can see the usefulness of these additional wildcard character through the table.

Oracle

Microsoft SQL Server

SELECT * FROM STUDENT_ADMIN.STUDENT
WHERE LNAME LIKE 'A%'
OR LNAME LIKE 'B%'
OR LNAME LIKE 'C%'

SELECT * FROM STUDENT_ADMIN.STUDENT
WHERE LNAME LIKE '[ABC]%'

[^] Wildcard character set is used to specify the characters NOT in the specified range. For example, when all the characters except for f to be allowed from a LIKE - Use the '[^ a f]' or LIKE '[^ abcdef]'.
