Purpose of Java

JAVA is not a scripting language used to quickly execute a few commands.

It is aimed at a language for large-scale program development that benefits when structured as classes and packages.



Run JAVA

JAVA execution takes two steps.

(1) Compile the Java source with the javac command and save it in the class file. (Create a .class file)

(2) Run the virtual machine with the java command and load the class file (.class) to execute the bytecode.

JAVA operation issue

If you want to normalize the number between 0 and 11 after adjusting the position of the hour hand, you can simply obtain (position + adjustment)% 12. However, if the adjustment is made negative, the value goes down to negative. So, we need to introduce a branch or (position + adjustment)% 12 + 12)% 12. But both of them are uncomfortable.

Therefore, we use the method of Math.floorMod method.

Math.floorMod (position + adjustment, 12) always gives a value between 0 and 11.

* floorMod gives a negative value for the number of divisions. But this situation does not come out well.



Math / Numerical Type Conversion

One billion times three is calculated as -1294967296. It is overflowed because the range of int is over 200 million. So you have to use the catch function as an exception. Calling Math.multiplyExact (1000000000, 3)



The Basics of Number Type Conversions

If one of the operands is a double, make the other double

If one of the operands is a float, the other is made float

If one of the operands is a long, the other is made long

Otherwise, convert the two operands to int.

In particular, the conversion from small type to large type is always legal, but always be careful when converting from large type to small type. The fractional part may be discarded or only the last byte may be.



Logical operation issue

&&, || (and, or) The first thing to worry about when doing an operation is that if the condition before && is false, then the condition after the second is not evaluated and that || Note that if the preceding condition is true, then the second and subsequent conditions are not evaluated.
