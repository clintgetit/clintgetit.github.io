Git 이란 개발하는 과정이나 소스코드 파일 등을 관리하는 도구입니다.
     플젝이나 코드가 개발되어 온 과정, 역사에 대해서 볼 수 있으며, 특정한 시점으로 rollback이 가능하다는 점이 있습니다.


 Git의 설치는 다음 링크에서 설치할 수 있습니다.
 https://git-scm.com/downloads

Github 회원가입도 필요조건입니다. 다음 링크에서 회원 가입을 해줍시다.
https://github.com/join


Git의 필수적인 명령어가 3개가 있습니다. ( 커밋이란 히스토리의 한 단위를 의미합니다. )
  1. add : 커밋할 목록에 추가 
  2. commit : 커밋을 만듭니다.
  3. push : 현재까지의 커밋들을 Github에 밀어넣습니다.


알아두면 좋을 Git에서 사용하는 상태 확인 명령어입니다.
# git show
# git log
# git shortlog
# git diff
# git status
