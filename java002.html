Input and Output

system.out.println: passed to standard output stream and displayed on terminal

Since the system.in object has only one method that reads only one byte, you need to create a Scanner linked to System.in.

Scanner in = new Scanner (System.in); // java.util.Scanner

Method for reading next line nextLine () / method for dividing by whitespace next () / method for reading integer nextInt () / method for reading floating point nextDouble ()

Use haNextLine, hasNext, hasNextInt, hasNextDouble to check for another line, word, integer, or floating point



Keeping passwords in the terminal

The Scanner uses the Console class because it is visible when a password is entered.

Console terminal = System.console ();

String username = terminal.readLine ("User Name:");

char [] passwd = terminal.readPassword ("Password:");



Arrays and Array Lists

You need the new operator to initialize the array.

You can get the length of the array with the array name .length.

When you create an object array that is not a base type, you must also initialize it.

ArrayList = Generic ArrayList

A wrapper class for the base type ArrayList <Integer> numbers = new ArrayList <> ();

Note that the conversion between the base type and the wrapper type is the only one. You should not use == as in if (numbers.get (i) == numbers.get (j)), and use equals to compare and check.



Copy from array

Shallow copy

int [] numbers = primes;

numbers [2] = 1;

If primes also refer to the same array. Thus primes [2] change as well.

If you do not want this sharing, use the static method Arrays.copyOf.

Deep copy

int [] copiedPrimes = Arrays.copyOf (primes, primes.length);



Copy from array list

Shallow copy

ArrayList <String> people = friends;

people.set (0, "Mary"); // friends.get (0) is also "Mary".

Deep copy

Arraylist <String> copiedFriends = new ArrayList <> (friends);

To copy, you must create a new array list from the existing array list.

* If you want to copy an array to an array list, you can use the ArrayList constructor.

However, you must enclose the array in a list with Arrays.asList.

ArrayList <String> friends = new ArrayList <> (Arrays.asList (names)); // names is an array



Array algorithm

The Arrays and Collections classes provide algorithm implementations.

Arrays.fill (numbers, 0); // int []

Collections.fill (friends, ""); // ArrayList <String>

Arrays.sort (names);

Collections.sort (friends);
